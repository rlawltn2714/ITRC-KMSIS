## Problem Statement
수도권 인구집중으로 건물 밀집도가 증가함에 따라 일조권 문제가 발생하고 있다. 이를 해결하기 위해, 특정 건물이나 위치의 일조량 관련 정보를 제공하는 시스템이 필요하다. 국토의 공간 정보와 건물 데이터가 3D 모델링 되어있어 시각적으로 일조량 관련 정보를 확인할 수 있고, 특정 건물이나 지역을 검색하여 정보를 열람할 수 있다. 사용자가 직접 만든 3D 모델 파일을 불러와 모의 건물을 설치하고, 일조량 변화를 예측할 수 있어야 한다. 또한 모든 UI는 유저친화적으로 일반인들도 쉽게 조작할 수 있어야 한다.
